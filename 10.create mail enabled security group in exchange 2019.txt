10. Create Mail-Enabled Security Group in Exchange 2019

Using Exchange Admin Center

Open Exchange Admin Center
Click on Recipients
Click on Groups
Click on + (Add)
Select Mail-Enabled Security
Enter the group name and alias
Add members to the group
Click Save

Using Exchange Management Shell

New-DistributionGroup `
-Name "IT-Support2" `
-Alias "IT-Support2" `
-Type Security `
-OrganizationalUnit "harethsoft.lan/Maddina/IT-Support" `
-ManagedBy "administrator@harethsoft.lan" `
-MemberJoinRestriction Closed `
-MemberDepartRestriction Closed

Add-DistributionGroupMember -Identity "IT-Support2" -Member "it1@harethsoft.lan"
Add-DistributionGroupMember -Identity "IT-Support2" -Member "it2@harethsoft.lan"
Verify the group

Get-DistributionGroup "IT-Support2" | fl Name, Alias, RecipientType

Get-DistributionGroupMember -Identity "IT-Support2" | fl Name, RecipientType
