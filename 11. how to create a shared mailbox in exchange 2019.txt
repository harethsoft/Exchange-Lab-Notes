11. Create Shared Mailbox in Exchange 2019

Using Exchange Admin Center

Open Exchange Admin Center
Click on Recipients
Click on shared tab
Click on + (Add)

Enter display name and email address
Click Save

Using Exchange Management Shell

```
New-Mailbox `
-Shared `
-Name "General Information2" `
-Alias "contact2" `
-OrganizationalUnit "harethsoft.lan/Maddina/IT-Support" `
-Database "DB01"

Add-MailboxPermission -Identity "contact2" -User "it1@harethsoft.lan" -AccessRights FullAccess
Add-MailboxPermission -Identity "contact2" -User "it2@harethsoft.lan" -AccessRights FullAccess

Add-MailboxPermission -Identity "contact2" -User "it1@harethsoft.lan" -AccessRights SendAs
Add-MailboxPermission -Identity "contact2" -User "it2@harethsoft.lan" -AccessRights SendAs
```

Verify the shared mailbox

Get-Mailbox "contact2" | fl Name,Alias,PrimarySmtpAddress,RecipientTypeDetails
