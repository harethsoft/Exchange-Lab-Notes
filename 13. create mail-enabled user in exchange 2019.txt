13. Create Mail-Enabled User in Exchange 2019

Using Exchange Management Shell

New-MailUser `
-FirstName "Najee" `
-LastName "Saad" `
-Name "Najee Saad" `
-DisplayName "Najee Saad" `
-Alias "Najee" `
-ExternalEmailAddress "najee.saad@hotmail.com" `
-UserPrincipalName "najee.saad@harethsoft.lan" `
-OrganizationalUnit "harethsoft.lan/Maddina/contacts" `
-Password (ConvertTo-SecureString 'P@ssw0rd' -AsPlainText -Force)

Get-MailUser najee.saad | fl Name,Alias,ExternalEmailAddress,UserPrincipalName
